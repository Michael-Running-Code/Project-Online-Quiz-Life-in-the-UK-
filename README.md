# Life in the UK Practice Quiz

Online practice tool for the official **Life in the UK** test. 

## Features
- 24 questions (randomized order and options)
- 45-minute timer
- Progress tracker (1–24 blocks)
- Jump to any question
- Detailed result page with correct/incorrect answers
